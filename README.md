# Volunteer Management System

A Volunteer Management System developed for NayePankh Foundation using Spring Boot, MySQL, HTML, CSS and JavaScript.

## Features
- Volunteer Registration
- View Volunteers
- Edit Volunteer Details
- Delete Volunteers
- Search Volunteers
- Responsive UI

## Technologies Used
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL
- HTML
- CSS
- JavaScript
